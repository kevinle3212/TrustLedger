# Canonical TrustLedger Deployment URL

**Authors & Contributors:** [Kevin Le](https://www.linkedin.com/in/lekevin1),
[Kellen Snider](https://www.linkedin.com/in/kellen-snider-683396256/)

Use `https://trustledger-zeta.vercel.app/en` as the canonical deployed frontend
URL for TrustLedger references and checks. Do not reintroduce the old
`src-trustledger.vercel.app` hostname in docs, agent guidance, or deployment
notes unless the user explicitly asks for historical context.
