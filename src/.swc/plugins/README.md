SWC and Next.js may create platform-specific plugin cache directories here, for
example `macos_aarch64_*`. Those directories are generated local state and are
not source configuration.

Commit only documentation or explicit project config. Do not commit native
plugin binaries, generated cache output, or machine-specific folders.
