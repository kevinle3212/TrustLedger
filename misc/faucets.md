# Sepolia ETH Faucets

Quick reference for getting test ETH on Ethereum Sepolia (chain ID 11155111).

| Faucet      | URL                                                                 | Notes                            |
| ----------- | ------------------------------------------------------------------- | -------------------------------- |
| Alchemy     | <https://www.alchemy.com/faucets/ethereum-sepolia>                  | Free with account, 0.5 ETH/day   |
| Infura      | <https://www.infura.io/faucet/sepolia>                              | Free with account                |
| Google      | <https://cloud.google.com/application/web3/faucet/ethereum/sepolia> | Google account required          |
| Chainlink   | <https://faucets.chain.link/sepolia>                                | Requires LINK balance on mainnet |
| QuickNode   | <https://faucet.quicknode.com/ethereum/sepolia>                     | Free, no account needed          |
| Sepolia PoW | <https://sepolia-faucet.pk910.de>                                   | Mine PoW in browser, no account  |

## Sepolia block explorers

- Etherscan: <https://sepolia.etherscan.io>
- Blockscout: <https://eth-sepolia.blockscout.com>

## Useful RPC endpoints (public, no key required)

These are rate-limited and unsuitable for scripts — use Alchemy/Infura for
`SEPOLIA_RPC_URL`.

```text
https://rpc.sepolia.org
https://ethereum-sepolia-rpc.publicnode.com
```
