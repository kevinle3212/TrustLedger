This file records source-confirmed project history. Use `Unreleased` for changes

## Unreleased

- Added client-proposed escrow flow with freelancer acceptance and client
  funding.
- Added ERC-20 escrow support through a permanent token allowlist.
- Added warranty holdback claims and deadline-based claim paths.
- Added arbitration commit-reveal voting, appeals, juror rewards, and slashing.
- Added juror staking, eligibility, cooldown, and reputation tracking.
- Added participant ratings and recovery tracking in `ReputationRegistry`.
- Added a Next.js frontend with Reown AppKit, wagmi, viem, i18n, email flows,
  IPFS upload support, notifications, and Playwright tests.
- Added GitHub Actions workflows for CI, deployment, docs, wiki sync, security
  scanning, React Doctor, GitHub Models, and Dependabot automerge.
